# Theory Archive | Архивы теоретических материалов

Данный репозиторий представляет собой структурированную базу знаний по теоретическим дисциплинам, изучаемым на механико-математическом факультете МГУ, а также по направлениям Machine Learning и Data Science.

## 📌 Направления
Архив разделен на ключевые блоки, охватывающие математический фундамент и прикладные области:

### 1. Математический фундамент (Mathematics)
- **Теория вероятностей и матстатистика:** Конспекты по распределениям, проверке гипотез и предельным теоремам.
- **Методы оптимизации:** Теоретические основы градиентного спуска, условия Каруша — Куна — Таккера ($KKT$).
- **Численные методы:** Сходимость и устойчивость разностных схем, методы решения УрЧП.

### 2. Machine Learning & Data Science
- **Classical ML:** Математическое обоснование алгоритмов (SVM, Decision Trees, Boosting).
- **Deep Learning:** Теория обратного распространения ошибки, архитектуры нейронных сетей и функции активации.
- **Интерпретируемость:** Теоретические аспекты SHAP и важности признаков.

### 3. Аэромеханика и Физика
- Теоретические основы механики сплошных сред.
- Математические модели вязкопластичности и динамики жидкости.

## 🛠 Технологический контекст
В материалах архива теоретические выкладки часто сопровождаются примерами реализации на:
- **Python** (библиотеки NumPy, Pandas, Scikit-learn) для проверки математических моделей.
- **SQL** (теория реляционных БД, оптимизация запросов).
- **LaTeX** (оформление всех теоретических справок и формул).

## 📂 Структура
Каждый раздел содержит PDF-файлы с конспектами, исходники в формате `.tex` или Jupyter-ноутбуки для интерактивного изучения теории.
